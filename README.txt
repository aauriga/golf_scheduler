# golf_scheduler
1. Install python-3.11.4-amd64.exe
2. Open CMD terminal
3. Type python --version and hit Enter
    Put downloaded and extract package under C:\Users\Administrator
4. Verify it shows Python 3.11.4
8. type 'cd golf_scheduler'
9. Type 'pip install -r requirements.txt' wait for installation to get done
10. Type 'py manage.py runserver'
11. Go to http://127.0.0.1:8000/ in browser
