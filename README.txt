# golf_scheduler
1. Put downloaded and extract package under C:\Users\`Your User Name`
2. Install python-3.11.4-amd64.exe(Included in the Package) (click Add python.exe to PATH and click Install Now)
3. Open CMD terminal (Hit Windows Key, and search in the box for CMD)
4. Type `python --version` and hit Enter
5. Verify it shows Python 3.11.4
6. Type 'cd golf_scheduler' and hit Enter
7. Type 'pip install -r requirements.txt' and hit Enter, wait for installation to get done
8. Type 'py manage.py runserver' and hit Enter
9. Go to http://127.0.0.1:8000/ in browser
10. If it shows Port Not accessible, please use `py manage.py runserver 127.0.0.1:7000`
